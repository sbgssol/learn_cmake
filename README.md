
# practice_cmake
This repo is used to store all CMake practicing code/examples

#### To build with ninja:
- Command: `$ cmake -GNinja ..`
- Release files: 
	1. `build.ninja`
	2. `cmake_install.cmake`
	3. `CMakeCache.txt`
	4. `rules.ninja`
